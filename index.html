The Tech Stack

  • Go and gRPC are used for building scalable and efficient APIs that handle communication between services.

  • DeepFace and Python are leveraged for facial recognition and processing, enabling accurate face embeddings.

  • AWS provides the scalable infrastructure necessary to host and manage our application efficiently.

  • Kubernetes orchestrates our containerized applications, ensuring high availability and scalability.

  • Milvus serves as our vector database, enabling efficient storage and retrieval of high-dimensional face embeddings.

  • TailWind CSS provides utility-first classes that streamline our styling process, ensuring a consistent and responsive design.
